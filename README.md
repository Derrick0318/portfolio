# Derrick's Portfolio

A Flask portfolio website for presenting four academic Computer Science projects in one place.

## Projects

- Crop Yield AI
- 168 Park Selayang
- M4Food
- UTSmartBot

## Local Preview

```bash
pip install -r requirements.txt
python app.py
```

Open `http://localhost:5173/`.

## Notes

- Each project has its own page and demo route.
- Project assets are organized under `projects/`.
- Sensitive files, compiled Python files, database files, model files, and local deployment folders are ignored by Git.
